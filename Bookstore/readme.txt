Проект с использованием c++, plpgsql, qt, представляющий собой взаимодействие с базой данных для книжного магазина.
На первой вкладке находится представление популярных книг и издателей в виде графиков, на второй - управление,
выдача книг, добавление пользователей, просмотр долгов пользователей.
Скрипты plpgsql в папке bin/scripts
Исполняемый файл bin/rpbd.exe
