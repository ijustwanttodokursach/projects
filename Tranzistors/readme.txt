Проект на c++, qt, с использованием Boost - random, представляющий собой модель замены транзисторов. Даны
3 типа с максимальным временем использования - 20, 30 и 40 дней, с каждым днём вероятность выхода транзистора
из строя увеличивается. На выбор предоставлена замена только одного и сразу всех, с целью вычисления затрат.
Изначально представлено соотношение максимального времени работы в виде круговой диаграммы, далее необходимо
ввести количество дней, которые будет работать система, на графике отображаются вероятности поломки транзисторов
каждый день.
Исполняемый файл - bin/mslast.exe
